# Mem0 — Scalable Long-Term Memory for Production AI Agents

## 基本信息

- **标题**: Mem0: Building Production-Ready AI Agents with Scalable Long-Term Memory
- **作者**: Prateek Chhikara, Dev Khant, Saket Aryan, Taranjeet Singh, Deshraj Yadav
- **日期**: 2025-04
- **链接**: https://arxiv.org/abs/2504.19413
- **领域**: cs.CL

## 核心方法

- **基础架构**: 以内存为中心，动态提取 + 整合 + 检索对话信息
- **图增强变体**: 基于图的记忆表示，捕捉关系结构
- **评估结果**: LOCOMO benchmark 上相比 OpenAI 方案 +26% (LLM-as-Judge), P95 延迟 -91%, Token 成本 -90%
